# プライバシーポリシー

最終更新日：2026年6月20日

Mindseeker（以下「本サービス」といいます。）は、利用者の個人情報および利用情報の保護を重要な責務と考え、以下のとおりプライバシーポリシーを定めます。

## 1. 取得する情報

本サービスは、以下の情報を取得する場合があります。

### 1.1 アカウント情報

* 氏名
* メールアドレス
* Googleアカウント情報
* プロフィール画像

### 1.2 利用者が入力する情報

* ゴール
* タスク
* 課題（Issue）
* メモ
* チャット内容
* 添付ファイル
* その他利用者が本サービスに登録する情報

### 1.3 自動的に取得する情報

* IPアドレス
* ブラウザ情報
* デバイス情報
* 利用日時
* 操作履歴
* Cookie等の識別子

## 2. 利用目的

取得した情報は、以下の目的で利用します。

* 本サービスの提供および運営
* ユーザー認証
* AIによる支援機能の提供
* お問い合わせ対応
* サービス改善
* 障害調査およびセキュリティ対策
* 利用規約違反への対応
* 新機能や重要なお知らせの通知

## 3. AIサービスの利用

本サービスでは、AI機能の提供のため、第三者が提供するAIサービスを利用する場合があります。

利用者が入力した情報の一部が、AIによる回答生成のために以下のサービスへ送信される場合があります。

* OpenAI
* Anthropic
* Google
* Microsoft Azure OpenAI Service
* その他当社が利用するAIサービス提供事業者

利用者は、機密情報や個人情報を入力する際、自らの責任において行うものとします。

## 4. 外部サービスの利用

本サービスでは以下の外部サービスを利用する場合があります。

* Google Authentication
* Supabase
* Vercel
* OpenAI
* Microsoft Azure
* その他本サービス運営上必要なクラウドサービス

これらのサービス提供事業者が、それぞれのプライバシーポリシーに基づき情報を取り扱う場合があります。

## 5. 第三者提供

当サービスは、以下の場合を除き、利用者の個人情報を第三者へ提供しません。

* 利用者の同意がある場合
* 法令に基づく場合
* 人の生命、身体または財産の保護のために必要な場合
* 不正利用の防止またはセキュリティ確保のため必要な場合

## 6. データの保管と安全管理措置

### 6.1 保管場所

利用者情報は、当サービスまたは委託先クラウドサービス事業者のサーバーに保管されます。

### 6.2 技術的安全管理措置

当サービスは、情報漏えい、紛失、改ざん等を防止するため、以下の措置を講じています。

* **Row Level Security (RLS)**: データベース層で利用者ごとにデータを分離し、他の利用者のデータへのアクセスを技術的に防止
* **保存時暗号化**: データベースストレージは AES-256 で暗号化
* **通信の暗号化**: すべての通信を HTTPS/TLS で保護
* **認証トークンの検証**: すべての API リクエストで認証済みユーザーの確認を実施
* **最小権限の原則**: アプリケーションから利用可能なデータベース操作を必要最小限に制限

### 6.3 運営者のデータアクセスについて

運営者は、以下の原則に基づき利用者のデータを取り扱います。

* サービス提供に必要な範囲を超えて、利用者のゴール、タスク、チャット等のコンテンツデータを閲覧しません
* システム障害の調査、セキュリティインシデントへの対応、不正利用の防止等、正当な業務上の理由がある場合に限り、必要最小限のデータにアクセスする場合があります
* アクセスが行われた場合は、操作ログ（application_logs）に記録されます

### 6.4 データの削除

利用者がアカウントを削除した場合、当該利用者に紐づくすべてのデータは、データベースのカスケード削除機能により自動的に削除されます。

## 7. Cookie等の利用

本サービスは、利便性向上や利用状況分析のためCookieその他の類似技術を利用する場合があります。

利用者はブラウザ設定によりCookieを無効化できますが、一部機能が利用できなくなる場合があります。

利用者は、法令の定めに従い、自身の個人情報について以下を請求することができます。

* 開示
* 訂正
* 追加
* 削除
* 利用停止

## 9. 未成年者の利用

未成年者が本サービスを利用する場合は、保護者の同意を得た上で利用してください。

## 10. プライバシーポリシーの変更

本サービスは、必要に応じて本ポリシーを変更することがあります。

重要な変更がある場合は、本サービス上で通知します。
